Back-end do projeto integrador Univesp.

É um projeto voltado para o desenvolvimento de um software que tem como foco a criação de um estoque.

O projeto deve ser feito utilizando:
- Framework web
  - Será utilizado o framework Spring.
- Banco de dados (noções)
  - Será utilizado o MySQL.
- Controle de Versão.
  - O próprio github.
